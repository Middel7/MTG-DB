================================================================================
LIENS ENTRE TABLES
================================================================================

  De                                         Vers                          Type
  ------------------------------------------------------------------------------
  scryfall_card_printings.card_id         →  scryfall_cards.id          FK CASCADE
  scryfall_card_printings.set_code        →  scryfall_mtg_sets.code     FK SET NULL
  scryfall_card_printings.cardmarket_id   →  cardmarket_products        référence simple
                                             .id_product                (propagé à toutes
                                                                         les langues via
                                                                         card_id+set_code+
                                                                         collector_number)
  scryfall_card_faces.card_id             →  scryfall_cards.id          FK CASCADE
  scryfall_card_prices.printing_id        →  scryfall_card_printings.id FK CASCADE
  cardmarket_price_guide_entries          →  cardmarket_products        FK SET NULL
    .id_product                              .id_product
  cardmarket_price_guide_entries          →  cardmarket_import_files    FK CASCADE
    .import_file_id                          .id

SCRIPTS D'IMPORT
================================================================================

  scripts/import_scryfall.py               Import Scryfall (cartes, éditions,
                                           prix Scryfall, cardmarket_id,
                                           printed_name) — source : all_cards
                                           Étape finale : propagation automatique
                                           de cardmarket_id aux impressions non-
                                           anglaises (même carte, même édition,
                                           même collector_number)
  scripts/import_cardmarket_products.py    Import Product Catalog Cardmarket
  scripts/import_cardmarket_price_guide.py Import Price Guide Cardmarket
  scripts/import_cardmarket_all.py         Import Cardmarket complet
  scripts/link_cardmarket_to_scryfall.py   Rapport de liaison CM <-> Scryfall
  scripts/import_game_changers.py          Mise à jour flag game_changer
